# Bank Telemarketing Campaign: Comparing Classifiers
📋 Overview
The current analysis revolves around enhancing the efficiency and conversion rates of a Portuguese retail bank's outbound telemarketing campaigns regarding term deposits. Through examining the past marketing information data, comprising approximately 
41,000 data entries based on 20 different socio-economic characteristics of clients, we compare the effectiveness of four basic machine learning classifiers, including logistic regression, K-nearest neighbors (KNN), decision trees, and support vector machines (SVM).
